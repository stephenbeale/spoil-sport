Spoil Sport — Privacy Policy

Effective date: 16 March 2026

Overview

Spoil Sport is a Chrome extension that hides sports spoilers from your YouTube feed. It is designed with privacy as a core principle — no data is collected, stored, or transmitted to any external server.

Data collection

Spoil Sport does not collect any personal data. Specifically:

  • No analytics or tracking scripts
  • No cookies (beyond Chrome's own storage API)
  • No external network requests
  • No user accounts or sign-in
  • No browsing history recorded

Data storage

The extension stores your preferences (blocked keywords, blocked channels, and enabled/disabled state) using Chrome's built-in chrome.storage.sync API. This data:

  • Stays within your Chrome browser profile
  • Syncs across your signed-in Chrome devices (if Chrome Sync is enabled)
  • Is never sent to any third-party server
  • Can be reset to defaults at any time from the extension's settings page

Permissions

Spoil Sport requests the following Chrome permissions:

  • storage — to save your keyword and channel preferences
  • scripting — to count hidden videos shown in the popup
  • activeTab — to interact with the current YouTube tab
  • Host permission (youtube.com) — to run the content script that hides spoilers on YouTube pages

Third-party services

Spoil Sport does not integrate with, send data to, or receive data from any third-party services.

Changes to this policy

If this privacy policy is updated, the new version will be published at this same URL. The effective date at the top of the page will be updated accordingly.
